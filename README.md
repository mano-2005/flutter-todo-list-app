# To-Do List App

A simple Flutter To-Do List application using SharedPreferences for local storage.

## Features

- Add tasks
- Display tasks in a ListView
- Save tasks locally using SharedPreferences
- Retrieve tasks after app restart
- Uses setState for state management

## Technologies Used

- Flutter
- SharedPreferences

## Project Structure

lib/
├── main.dart
└── todo_screen.dart

## Run the App

flutter pub get

flutter run
